Разархивировать архив в папку.

В командной строке перейти в эту папку  и ввести команду: 
docker-compose up -d --build. 

Результат работы можно посмотреть в логе Docker Desktop или через терминал.
Команда:
docker ps -a #определить ID контейнера, выполняющего скрипт - [CONTAINER ID]
Посмотреть логи:
docker logs [CONTAINER ID]
